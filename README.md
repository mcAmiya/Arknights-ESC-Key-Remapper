# 明日方舟 ESC 键绑定修改工具（Arknights-ESC-Key-Remapper）

> 一键修改《明日方舟》PC 版中 **ESC 功能键** 的绑定，无需手动编辑注册表！

## 警告！本脚本会修改注册表，操作前请先备份您的注册表，因使用本脚本导致的损失与本项目以及项目贡献者无关！若您继续使用本脚本，则表示同意风险自负！

## 运行
1. `win` + `r` 弹出运行窗口
2. 输入 `powershell`
3. 在 `powershell` 中输入 `irm https://raw.githubusercontent.com/mcAmiya/Arknights-ESC-Key-Remapper/refs/heads/main/Arknights-ESC-Key-Remapper.ps1 | iex` 回车执行
